1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and
structure web pages using elements like headings,paragraphs, images, and links.

2. What is the difference between HTML and XHTML?

  • HTML is flexible with rules.
  • Tags in HTML can be left open sometimes.
  • HTML is not case-sensitive; XHTML is.
  • XHTML is stricter and follows XML rules.
  • All tags must be closed and written in lowercase.
  • In XHTML, tags must be closed and written in lowercase.

3.What is a tag?

A tag is a special code used in HTML to define elements on a web page.
Tags are enclosed in angle brackets, like ``, and usually come in pairs: an opening tag and a closing tag.

4. What are elements?

Elements are the building blocks of HTML documents.They consist of a start tag, content, and an end tag. For example, `

This is a paragraph.

` is a paragraph element.

5. What is the difference between HTML and HTML5?

  • HTML5 is the latest version of HTML.
  • HTML5 supports audio, video, and animation tags directly.
  • It is more mobile and modern browser-friendly.
  • HTML5 allows local storage and geolocation features.
  • HTML (older versions) don’t support these advanced features.

6. What are lists?

Lists are used to display multiple items.
Types of lists in HTML:
  • Ordered list (ol) – with numbers
  • Unordered list (ul) – with bullets
  • Description list (dl) – with terms and descriptions

7. What is the difference between XML and HTML?

  • XML (eXtensible Markup Language) is a markup language designed to store and transport data, while HTML (HyperText Markup Language) is used to display data on web pages.
  • XML is strict about syntax and requires all tags to be closed, while HTML is more lenient.
  • XML allows users to define their own tags, whereas HTML has predefined tags.
  • XML is primarily used for data interchange, while HTML is used for creating web content.

8. What are attributes?

  • Attributes provide additional information about HTML elements.
  • They are specified within the opening tag of an element and consist of a name and a value, like ``.
  • Common attributes
    • id: Used to assign a unique identifier to an element.
    • src: Specifies the source of an image or media file.
    • href: Specifies the URL of a link.
